### 🎥 Introduction

**YT Premium Ultra** is a powerful Android application built with **Flutter** that delivers a complete YouTube experience **without advertisements**, interruptions, or the need for a Premium subscription.

Using an advanced **JavaScript Shield Engine**, the app effectively blocks all types of ads while maintaining smooth video and music playback even when the app is minimized or the screen is off.

---

## ✨ KEY FEATURES

- **Complete Ad Elimination**: Removes pre-roll, mid-roll, banners, overlays, sponsored cards, and promoted content
- **Immortal Background Playback**: Continue listening to music even when the screen is off or you switch to another app
- **Self-Healing Playback**: Automatically resumes playback if YouTube tries to interrupt or pause the video
- **Battery & RAM Efficient**: Deeply optimized thanks to lightweight WebView and smart JavaScript injection
- **Clean & Modern Interface**: Default dark mode with full immersive fullscreen support
- **Smart Adaptation**: Smooth screen rotation handling and edge-to-edge display
- **Anti-Tracking Protection**: Reduces YouTube’s telemetry and tracking data collection
- **Fast Launch**: Beautiful animated splash screen

---

## 🧠 CORE TECHNOLOGIES

**YT Premium Ultra** is powered by modern and robust technologies:

- **Flutter** – Smooth UI and high performance
- **webview_flutter** – Powerful WebView with unrestricted JavaScript support
- **Custom Ad-Blocking Engine** – Multi-layered JavaScript (Fetch hook, XHR override, MutationObserver, CSS Shield)
- **Background Persistence** – Keeps video playing even when the app is in the background
- **Anti-Detection Patches** – Simulates continuous visibility state

## 📥 INSTALLATION GUIDE

### Method 1: Download APK (Recommended)
1. Go to the **Releases** tab of this repository
2. Download the latest `app-YT-Premium-Ultra.apk` file
3. Install on your Android device (enable "Unknown Sources" if prompted)

### Method 2: Build from Source
```bash
git clone https://github.com/ten-ban/YT-Premium-Ultra.git
cd YT-Premium-Ultra
flutter pub get
flutter build apk --release
